
# Soluciones del Laboratorio 3 — ARSW

En cada archivo ZIP incluido junto a este repositorio se encuentra la solución correspondiente a una de las partes del laboratorio. Cada ZIP contiene el código fuente, instrucciones para ejecutar la parte y evidencias o resultados asociados.

Cómo identificar cada ZIP
- Use el nombre del archivo ZIP para identificar la parte correspondiente (por ejemplo, `part1.zip` → Parte I, `part2.zip` → Parte II, `part3.zip` → Parte III).

Contenido de cada ZIP
- Código fuente completo
- Instrucciones de compilación y ejecución
- Capturas o logs que evidencian el correcto funcionamiento
- Informe de la parte: cada ZIP incluye un `README` con el informe y resultados específicos de esa parte del laboratorio
